To-Do List Interativa em JavaScript

Este projeto foi desenvolvido para criar uma lista de tarefas interativa utilizando apenas JavaScript. A ideia surgiu da necessidade de organizar melhor minhas atividades diárias e praticar conceitos fundamentais da linguagem, como manipulação do DOM, eventos e armazenamento local com localStorage.

Ao longo do desenvolvimento, implementei funcionalidades como adicionar, remover, marcar tarefas como concluídas e manter os dados salvos mesmo após fechar o navegador. O visual foi mantido simples e objetivo, com foco na usabilidade e clareza.

Esse projeto foi uma ótima oportunidade para colocar em prática os conhecimentos adquiridos e entender na prática como o JavaScript pode tornar uma aplicação web dinâmica e funcional. Com o tempo, pretendo evoluí-lo ainda mais, adicionando categorias, filtros e integração com banco de dados.

Além de ser uma ferramenta útil para o dia a dia, este projeto também representa meu compromisso com a melhoria contínua e meu entusiasmo em transformar ideias em soluções práticas por meio da programação.
